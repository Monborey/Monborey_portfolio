# MON Borey – Portfolio

A clean, minimalist portfolio built with React + Tailwind CSS.

## Tech Stack
- React 18
- Tailwind CSS 3
- Vite 5


## Project Structure

```
src/
├── components/
│   ├── Navbar.jsx       # Fixed navigation bar
│   ├── Hero.jsx         # Landing / hero section
│   ├── About.jsx        # Personal info section
│   ├── Experience.jsx   # Work experience + certifications
│   ├── Education.jsx    # Academic background
│   ├── Projects.jsx     # Academic projects
│   ├── Skills.jsx       # Technical & soft skills
│   ├── Contact.jsx      # Contact + references
│   └── Footer.jsx       # Footer
├── data/
│   └── cv.js            # All CV data (edit here)
├── App.jsx
├── main.jsx
└── index.css
```

## Customization
All content lives in `src/data/cv.js`. Edit that file to update any personal information, experience, or skills without touching the UI components.
